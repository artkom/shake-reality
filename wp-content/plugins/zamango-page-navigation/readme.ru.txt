Плагин создает настраиваемую полосу навигации в списках, а также ссылки на предыдущий и следующий посты на странице поста.

== Description ==

Плагин [Zamango Page Navigation](http://www.zamango.com/pagenavi.htm "Блог Zamango") вставляет настраиваемые ссылки быстрой навигации по страницам, позволяя читателю, например, переходить на 12-тую страницу категории за один клик по линейке навигации, а также переходить к предыдущему или следующему посту.

Основные возможности:

* Создание линейки ссылок на странице списков (главная, страница категории, поиск по тагам, архивы и т.п.)
* Создание ссылок быстрого перехода к предыдущему и следующему посту
* Отображение линейки ссылок в начале и/или в конце страницы без правки темы
* Возможность настройки внешнего вида с помощью HTML тегов и CSS
* Настройка количества ссылок в левой, центральной и правой части линейки независимо друг от друга
* Другие настройки, позволяющие кастомизировать линейку ссылок под нужды разных сайтов

Основная причина, по которой был создан этот плагин - это отсутствие плагина подобной функциональности, который работал бы на современных версиях вордпреса, поэтому пришлось создать его для сайтов, собранных с помощью [плагина Zamango Money Extractor](http://wordpress.org/extend/plugins/zamango-money-extractor/). Однако, мы надеемся, что Zamango Page Navigation окажется полезным и другим пользователям Wordpress.

В классической теме есть быстрые ссылки на странице поста, но они по какой-то прихоти создателя используют нелинейную последовательность постов, так что дважды перейдя к "следующему посту", а потом дважды перейдя к "предыдущему посту" читатель скорее всего окажется не на той странице, откуда он начал. Вроде бы мелочь, но внутреннюю перелинковку сайта и перераспределение пейджранка нарушает.


== Installation ==

Процесс установки стандартен:

1. Распакуйте архив с плагином на свой компьютер
1. Залейте получившуюся папку `zamango-page-navigation` в директорию `/wp-content/plugins/` вашего сайта
1. Активируйте плагин на странице 'Plugins' админки вордпреса


== Frequently Asked Questions ==

= Будет ли плагин работать на более старой версии вордпреса, чем 2.8? =

Скорее всего не будет, т.к. плагин использует некоторые возможности Wordpress, появившиеся только в версии 2.8. Поэтому рекомендуем обновиться до какой-нибудь более новой версии - они и умнее, и надежнее.
Хотя с некоторыми ограничениями он работает даже на Wordpress 2.7.4.

= Что, если я не хочу использовать ссылки быстрого перехода на странице поста? =

Просто снимите галочки с параметров "перед постом" и "после поста" на вкладке "Переходы между постами".

= Тема, которую я использую, уже отображает ссылки на следующую страницу =

Чтобы не изменять вручную темы предлагаю просто добавить в CSS плагина строку вида
 `    .navigation {display:none;} `
