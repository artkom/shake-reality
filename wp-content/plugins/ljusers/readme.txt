== Description ==

Plugin for usefull reference to livejournal users.

* Use [ljuser]USERNAME[/ljuser] for reference to USERNAME in livejournal.com.
* Use [ljuser name="OTHERNAME"]USERNAME[/ljuser] for reference to USERNAME in livejournal.com, but visitors will see "OTHERNAME" with link to USERNAME.livejournal.com (see screenshots)

* Use [ljcomm]COMMNAME[/ljcomm] for reference to community "COMMNAME" in livejournal.com
* Use [ljuser name="OTHERNAME"]COMMNAME[/ljuser] for reference to "COMMNAME"  in livejournal.com, but visitors will see "OTHERNAME" with link to community.livejournal.com/COMMNAME (see screenshots)

* Use [liruman]USERNAME[/liruman] for reference to USERNAME in liveinternet.ru (man).
* Use [lirugirl]USERNAME[/lirugirl] for reference to USERNAME in liveinternet.ru (girl).

* Use [liruman name="OTHERNAME"]USERNAME[/liruman] for reference to USERNAME (man) in liveinternet.ru, but visitors will see "OTHERNAME" with link to http://www.liveinternet.ru/users/USERNAME/profile/ (see screenshots)

* Use [lirugirl name="OTHERNAME"]USERNAME[/lirugirl] for reference to USERNAME (girl) in liveinternet.ru, but visitors will see "OTHERNAME" with link to http://www.liveinternet.ru/users/USERNAME/profile/ (see screenshots)

== Installation ==

1. Upload the 'ljusers' folder to the '/wp-content/plugins/' directory; and
2. Activate the plugin through the 'Plugins' menu in WordPress.
